How to use

For Nightlords (quick method):

  1. Have the boss visible in-game
  2. Select the boss name from "Quick save nightlord..." dropdown
  3. Click Save Boss (uses your calibrated boss region)

For Regions & Shifting Earth:

  1. Capture Screen - takes a screenshot of the map region
  2. Draw a box - click and drag around what you want
  3. Select what it is - choose from dropdown
  4. Save

Full Screen Mode - For Field Boss OCR:

  1. Switch to Full Screen mode above
  2. Have a field boss name visible on screen
  3. Capture Screen - captures the entire monitor
  4. Draw a box around the boss name text
  5. Select Field Boss Name and save

This region will be used for OCR to read field boss names during gameplay.

Restart OCR service after saving to apply changes.
